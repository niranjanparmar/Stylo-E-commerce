# 🛍️ Stylo E-commerce

**Stylo E-commerce** is an online shopping platform/brand that offers a variety of fashion products such as clothing and lifestyle items. Customers can browse, select, and purchase products easily with online payment and delivery options.

---

## 🚀 Features

- 🏠 Browse homepage with product listings  
- 🛒 Add products to cart  
- 🔍 View product details  
- 👤 User authentication (Sign up / Login)  
- 💳 Online payment support  
- 📦 Order review and checkout  
- 📱 Responsive UI for all devices  

---

## 💻 Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Database:** SQLite  
- **Version Control:** Git & GitHub
